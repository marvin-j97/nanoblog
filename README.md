# nanoblog

Lightweight Astro blog template.

## Features

- Lightweight (about 7 kB per page)
- Fully responsive
- Light + dark mode
- Full-text search
- Featured posts
- Pagination
- Sitemap generation
- SEO stuff (meta & OG tags)
- RSS feed
- Reading time calculation & progress
- Supports blog thumbnail images
